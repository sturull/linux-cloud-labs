# Linux Basics

## Essential Commands
- `ls` – list files
- `cd` – change directory
- `mkdir` – create folder
- `rm` – delete file
- `cp` – copy files
- `mv` – move/rename files
- `nano` / `vim` – edit files
- `sudo apt update` – update packages (Ubuntu)
- `sudo yum update` – update packages (CentOS)

## Practice Tasks

2. Add 3 files inside it.
3. Move one file to a new folder.
